Raspberry Pi

Installation instruction

Login as alarm (alarm) and do initial set up. Most of the instructions are available on ArchWiki.

# su - root # password root # install terminus (forgot how) # setfont # nano /etc/vconsole.conf # add "KEYMAP=ter-132b" # ln -sf /usr/share/zoneinfo/Region/City /etc/localtime # hwclock --systohc didn't work on me # Does not seem to have a problem # nano /etc/locale.gen # uncomment en_US.UTF-8 # locale-gen # nano /etc/locale.conf # add one-liner "LANG=en_US.UTF-8" # nano /etc/hostname rpi4 # nano /etc/hostname 127.0.0.1 localhost ::1 localhost 127.0.1.1 myhostname.localdomain myhostname useradd -m nobiot passwd nobiot # enter password

Install sudo

pacman -S sudo nano /etc/sudoers # add new user passwd root # enter new password reboot

Log in as nobiot

sudo pacman -Syu sudo pacman -S i3 xorg-server xorg-apps # Take the default option for both sudo pacman -S micro feh git emacs man

Now [[i3]] works with a minimum modification to the default template that it generates at the first launch. Modify .bash_profile to start i3 automatically by adding: startx
